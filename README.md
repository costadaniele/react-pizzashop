# React + TypeScript + Vite

## Instalação PNPM

`npm install -g pnpm`

## Instalação React 

`pnpm create vite@latest`

`pnpm i`

## Instalando shadcn/ui

`pnpm add -D tailwindcss postcss autoprefixer`

`npx tailwindcss init -p`

https://ui.shadcn.com/docs/installation/vite
